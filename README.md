# 💬 Live Messaging App

A modern real-time messaging application inspired by popular chat platforms. The application enables users to communicate instantly through a clean and responsive interface with secure authentication, live messaging, and cloud-based data storage.

---

## 🚀 Features

- Secure user authentication using Google Sign-In
- Real-time one-to-one messaging
- Live chat synchronization
- Online conversation interface
- Persistent chat history using Firebase
- Responsive design for desktop and mobile devices
- Message timestamps
- User profile support
- Fast and intuitive user experience

---

## 🛠️ Tech Stack

### Frontend
- Vue.js 3

### Backend & Database

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Kashfakulsoom/Live-chat
```

### Navigate to the project

```bash
cd live-messaging-app
```

### Install dependencies

```bash
npm install
```

### Configure Firebase

Create a Firebase project and enable:

- Authentication (Google Sign-In)
- Cloud Firestore

Update the Firebase configuration inside the project with your credentials.

---

## ▶️ Run the Application

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
